# 🌱 AgriCentral  
### A Data-Driven Monitoring System for Equipment Utilization, Crop Production, and Livestock Management in Marinduque Provincial Agriculture

---

## 📌 Project Overview  
**AgriCentral** is a data-driven monitoring system designed to support the **Marinduque Provincial Agriculture Office** in managing and optimizing agricultural operations. The system focuses on improving the tracking and analysis of:

- 🚜 Equipment utilization  
- 🌾 Crop production  
- 🐄 Livestock management  

By centralizing agricultural data, AgriCentral enables better decision-making, resource allocation, and overall efficiency in provincial agricultural programs.

---

## 🎯 Objectives  
- To digitize and streamline agricultural data management  
- To monitor equipment usage and availability  
- To track crop production data across different municipalities  
- To manage livestock records efficiently  
- To provide data insights for planning and decision-making  

---

## 🛠️ Technologies Used  
- Frontend: React.js  
- Backend: Node.js / Express.js  
- Database: MongoDB  
- Deployment: Vercel
